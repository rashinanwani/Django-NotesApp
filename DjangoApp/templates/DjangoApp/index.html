Commands for Beginners:



  • Command to check whether python is installed or not: python --version
  • Command to check whether pip is installed or not: pip --version
  • Command to install django: pip install django
  • Command to check whether django is installed or not: import django print(django.get_version())
  • Command to start the new project: django-admin startproject app
  • Command to run the project: python manage.py runserver
  • Command to change the port no: python manage.py runserver 8080(by default it is 8000)
  • Command to create your app: python manage.py startapp app_name
  • Command to create migrations for those changes: python manage.py makemigrations app_name
  • Command to run the migrations and manage your database schema automatically: python manage.py migrate
  • Command to takes migration names and returns their SQL: python manage.py sqlmigrate app_name 0001
  • Command to create an admin user: python manage.py createsuperuser
